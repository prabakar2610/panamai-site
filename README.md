# PanamAI Multi-Product Website

This repository contains the PanamAI static website featuring 5 product sections:
1. Learn
2. Tools
3. Blog
4. Advisory
5. AI Hub

## Deploy
- Push to GitHub and enable GitHub Pages (root directory).
- Or deploy easily on Vercel/Netlify.

## Edit
Each `.html` file represents a standalone product page with placeholder content.
